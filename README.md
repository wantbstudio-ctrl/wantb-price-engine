# WantB Price Engine

WantB Price Engine은 셀러용 가격 계산 MVP입니다.

## 1. 설치

```bash
npm install
```

## 2. 실행

```bash
npm run dev
```

브라우저에서 아래 주소로 접속합니다.

```bash
http://localhost:3000
```

## 3. 아이폰 접속

PC와 아이폰이 같은 Wi-Fi에 연결된 상태에서,
PC의 IPv4 주소를 확인한 뒤 아래처럼 접속합니다.

```bash
http://PC_IP:3000
```

예시:

```bash
http://192.168.0.10:3000
```
